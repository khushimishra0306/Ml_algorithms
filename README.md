This repository contains implementations of various **Machine Learning algorithms**, in two ways:
1. Using popular Python libraries (such as scikit-learn,scipy,mlxtend,numpy,pandas etc.).
2. Implementing algorithms **from scratch** to understand the underlying mathematics intuition.

## 📌 Objectives
- Strengthen understanding of ML fundamentals by coding algorithms manually.
- Compare performance and efficiency between library-based and scratch implementations.
- Build a reference collection of ML algorithms for study and practice.
